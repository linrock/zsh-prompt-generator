Zsh prompt generator

Zsh prompt variables

These are the variable substitutions used above. 

%n	 The user that is logged in
%M	 Full hostname
%1~  Current directory, replacing the home dir with ~

Some more variable substitutions 

%d   Current directory (alternative: %/ )	a number following the “%” indicates how deep the displayed path will be
%l	 The TTY device you’re using, strips "/dev/" or "/dev/tty"
%m   Hostname up until the first dot. A number after "%m" will indicated how many dots should be included
%y	 Same as "%l" just does not strip “/dev” or “/dev/tty”

About

This page is intended to help you easily explore and create color schemes for your zsh prompt. The structure of the prompt itself is not customizable yet. If you want to do that, you can reference the zsh prompt variables above and manually edit PS1 with the variables you want.

This is a work in progress.
Source code: https://github.com/linrock/zsh-prompt-generator
